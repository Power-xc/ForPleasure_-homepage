# Objection Handling Bot Skill

> **담당:** Growth
> **Tier:** 1 (Claude Haiku)
> **Skill ID:** `objection`

---

## 언제 쓰는가

- "광고 클릭은 많은데 구매로 안 이어져"
- "장바구니 이탈률이 높아"
- "고객이 왜 안 사는지 이유를 모르겠어"
- "상세페이지에 반론 처리 문구가 빠진 것 같아"
- `@objection handle {제품/가격대/타겟}` 트리거 호출 시

> **핵심 원칙:** 구매 전환 병목의 80%는 3가지 반론에서 생긴다.
> "비싸요" / "필요없어요" / "나중에요"
> 이 세 가지를 페이지 안에서 먼저 해결하면 전환율이 올라간다.

---

## 실행 프로토콜

`@objection handle` 트리거가 오면 3대 반론 × 세부 유형별로 대응 카피를 생성한다.

---

### Objection 1 — "비싸요" (가격 저항)

세부 유형 3가지를 각각 처리한다.

```
① "그냥 비쌈" 유형 → 가치 재프레이밍
   공식: 일 단위 환산 + 비교 앵커
   예: "하루 커피 한 잔 값으로, {혜택}을 매일 누릴 수 있어요"
   예: "{경쟁사/대안} 대비 X% 저렴하면서 Y는 포함"

② "대비 가치를 모름" 유형 → 비용 vs 손해 대조
   공식: 안 샀을 때의 손실을 가시화
   예: "이걸 안 쓰면 {고통}에 드는 비용이 월 {금액}입니다"

③ "예산이 없음" 유형 → 결제 마찰 제거
   공식: 분할/무이자/환불보장 강조
   예: "부담 없이 시작: 3개월 무이자 할부 / 30일 환불 보장"
```

---

### Objection 2 — "필요없어요" (필요성 부정)

```
① "나한테 해당 안 됨" 유형 → 구체적 타겟 재설정
   공식: 이 제품이 누구를 위한 것인지 명확히 + 후기로 증명
   예: "이런 분께 딱입니다: {상황 3가지}"
   → 체크리스트 형식으로 2개 이상 해당하면 필요한 사람임을 보여줌

② "지금 당장 문제가 없음" 유형 → 잠재 손실 가시화
   공식: "지금은 괜찮아도, X개월 뒤 {결과}"
   예: "지금 대비 안 하면 {나쁜 결과}가 옵니다 — 실제 사례"

③ "혜택을 못 느낌" 유형 → Before/After 구체화
   후기 기반 수치 제시 + 무료 체험/샘플로 직접 느끼게
```

---

### Objection 3 — "나중에요" (결정 연기)

```
① "더 알아보고 결정" 유형 → 비교 우위 정리
   공식: 비교표 제공 (우리 vs 대안 3가지, 우리가 이기는 항목만)
   예: "이미 비교하셨나요? {경쟁사 이름} 대비 {우리만 있는 것}"

② "타이밍이 아님" 유형 → 지금이 최적 타이밍임을 증명
   공식: 실행 지연 비용 계산 + 사회적 증거
   예: "지금 시작한 고객의 X%가 첫 달에 {결과} 달성"

③ "귀찮음/리스크 두려움" 유형 → 마찰 최소화
   공식: 온보딩 3단계 이내 + 환불 보장 + 성공 사례
   예: "3분 설정, 마음에 안 들면 전액 환불 — 0 리스크"
```

### 출력 형식

각 반론에 대해 아래 형식으로 출력:

```markdown
### [반론 유형]
**반론 인식 문구:** (고객 입장에서 쓰는 말)
**대응 카피:** (페이지에 바로 넣을 수 있는 문장)
**배치 위치:** (가격표 아래 | CTA 위 | FAQ 섹션 등)
**강도:** 약 / 중 / 강 (이 반론이 실제 전환을 얼마나 막는지)
```

### A/B 테스트 제안

```
반론 처리 카피 적용 후:
→ 장바구니 이탈률 측정 포인트: 가격 섹션 스크롤 이탈
→ 클릭 측정 포인트: FAQ 섹션 클릭률
→ 유의미한 차이 기준: 전환율 +0.5%p 이상

→ A/B 테스트 설계: @funnel analyze
```

## Hard Gates

| ID | 조건 | FAIL 기준 |
|----|------|-----------|
| G1 | 3대 반론 포함 | 비쌈·필요없음·나중에 중 하나라도 누락 |
| G2 | 증거 기반 대응 | 주장만 있고 근거, 비교, 보증 중 하나도 없음 |
| G3 | 구매 전환 연결 | 대응 문구가 CTA 또는 다음 행동으로 이어지지 않음 |
