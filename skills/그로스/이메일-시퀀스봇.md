# 이메일 시퀀스봇 스킬 (Email Sequence Bot Skill)

> **담당:** 그로스부
> **Tier:** 1 (Claude Haiku)
> **Skill ID:** `email`

---

## 언제 쓰는가

- "이메일로 구매 유도하고 싶은데 시퀀스를 짜줘"
- "뉴스레터 가입자한테 보낼 웰컴 시리즈 만들어줘"
- "구매 후 재구매 유도 이메일 써줘"
- `@email sequence {제품/목표(신규전환|재구매)}` 트리거 호출 시

---

## 실행 프로토콜

`@email sequence` 트리거 후 목표(신규 전환 / 재구매)를 확인 후 해당 시퀀스 출력.

---

### Sequence A — 신규 전환 7단계 (구독 → 첫 구매)

각 이메일마다: 발송 타이밍 / 제목 / 핵심 메시지 / CTA / 오픈율 예측 포인트

```
D+0  [웰컴] — 환영 + 기대감 설정
     제목 공식: "드디어 오셨군요, {이름}님 — 먼저 이것부터 챙겨가세요"
     내용: 무엇을 받게 될지 예고 + 즉시 쓸 수 있는 작은 가치(무료 자료/할인코드) 제공
     CTA: 무료 자료 받기 / 가이드 보기
     오픈율 포인트: 발송 즉시 → 오픈율 최고점

D+2  [가치 제공 1] — 문제 공감 + 인사이트
     제목 공식: "왜 {타겟의 흔한 실수}를 반복하는 걸까요?"
     내용: Pain Point 심화 → 원인 분석 → 힌트만 제공 (해결책은 다음 이메일에)
     CTA: 이 글 읽기 (블로그/콘텐츠 링크)

D+4  [가치 제공 2] — 솔루션 공개
     제목 공식: "지난 이메일에서 말씀드린 해결책입니다"
     내용: 실제 사용법 또는 케이스 스터디 → 우리 제품이 자연스럽게 등장
     CTA: 더 알아보기

D+6  [신뢰 구축] — 사회적 증거
     제목 공식: "{고객 이름}님이 {결과}를 낸 방법"
     내용: 구체적 후기 1개 (Before/After + 수치) → 나도 할 수 있다는 감각
     CTA: 같은 결과 얻기

D+8  [구매 유도 1] — 소프트 CTA
     제목 공식: "이분들은 이미 시작하셨습니다"
     내용: 사용자 수/후기 수 사회적 증거 + 제품 소개 (판매 냄새 최소화)
     CTA: 지금 시작하기 (부드러운 버튼)

D+11 [구매 유도 2] — 혜택 강조
     제목 공식: "{이름}님께만 드리는 혜택이 있어요"
     내용: 특별 할인 or 보너스 + 유효기간 72시간
     CTA: 혜택 받기 (D+14 만료)

D+14 [마감 긴급성]
     제목 공식: "오늘 자정에 마감됩니다"
     내용: 혜택 마감 알림 + 반론 처리 (FAQ 1~2개) + 마지막 설득
     CTA: 지금 바로 시작하기
```

---

### Sequence B — 재구매 유도 4단계 (구매 후 → 재구매)

```
D+7  (첫 구매 후) [온보딩]
     "잘 받으셨나요? 이렇게 쓰면 더 좋아요"
     → 사용 팁 + 커뮤니티/채널 안내

D+21 [중간 체크인]
     "어떠세요? 이런 결과 나오셨나요?"
     → 성과 확인 + 사용자 후기 유도 + 업그레이드 힌트

D+45 [업셀/재구매 타이밍]
     "이 정도 쓰셨으면 이건 어때요?"
     → 다음 단계 제품/플랜 소개 (사용 데이터 기반 개인화)

D+60 [윈백 (미사용자)]
     "{이름}님, 오래 못 뵀어요"
     → 새로운 기능/혜택 + 복귀 인센티브
```

## 이메일 작성 원칙

```
제목 금지 패턴:
  ✗ "특가 안내드립니다" (스팸 필터 + 오픈율 저하)
  ✗ 느낌표 2개 이상
  ✗ 전부 대문자

제목 권장 패턴:
  ✓ 개인화 토큰 ({이름}님)
  ✓ 질문형 또는 불완전 문장 (호기심 유발)
  ✓ 30자 이내 (모바일 미리보기 기준)

본문 원칙:
  - 1이메일 1메시지 (CTA 1개만)
  - 읽는 데 90초 이하
  - 첫 문장이 두 번째 문장을 읽게 만들어야 함
```

## Hard Gates

| ID | 조건 | FAIL 기준 |
|----|------|-----------|
| G1 | 발송 타이밍 | 각 이메일의 발송 시점이 없음 |
| G2 | 제목/CTA 포함 | 제목 또는 CTA가 빠진 이메일이 있음 |
| G3 | 세그먼트 목적 | 신규전환/재구매 목적이 구분되지 않음 |

|----|------|------|
