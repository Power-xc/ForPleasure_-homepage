# 론칭 플래너 스킬 (Launch Planner Skill)

> **담당:** 그로스부
> **Tier:** 2 (Claude Sonnet)
> **Skill ID:** `launch`

---

## 언제 쓰는가

- "제품 출시 전 콘텐츠 캘린더 짜줘"
- "D-30부터 D-Day까지 뭘 올려야 해?"
- "리타겟팅 광고 어떻게 세팅해야 해?"
- "출시 직전 긴급성 만드는 방법 알려줘"
- `@launch plan {제품/출시일/채널/타겟}` 트리거 호출 시

---

## 실행 프로토콜

입력: `제품명 / 출시일 / 채널 / 타겟 / 핵심 차별점`

두 파트를 순서대로 실행한다: D-30 캘린더 → 리타겟팅 카피

---

### Part 1 — D-30 ~ D-Day 콘텐츠 캘린더

각 날짜별 출력 형식: `날짜 | 단계 | 콘텐츠 주제 | 핵심 문구 | 채널 우선순위`

```
[D-30 ~ D-21] 문제 공감 (Pain Point 빌드업)
목표: 우리 제품 언급 없이 Pain Point를 공유하는 사람들을 모은다
전략: 교육형/공감형 콘텐츠 → 팔로워/구독자 신뢰 확보
콘텐츠 유형: 카드뉴스, 짧은 영상, 커뮤니티 질문 게시글
핵심 문구 방향: "혹시 이런 경험 있으신가요?" / "이게 생각보다 많은 분들의 문제더라고요"

[D-20 ~ D-11] 해결책 힌트 (티저)
목표: "뭔가 나온다"는 기대감 형성, 사전 신청/알림 리스트 수집
전략: 솔루션 힌트 공개 + 사전 알림 신청 CTA
콘텐츠 유형: 비하인드 스냅샷, "곧 공개됩니다" 포맷, 설문
핵심 문구 방향: "저희가 준비하고 있어요" / "어떤 게 더 필요하세요?"

[D-10 ~ D-4] 기대감 (스포일러)
목표: 핵심 기능 1~2개 공개, 얼리버드 혜택 예고
전략: 기능 스포일러 + 사용 화면 일부 공개 + 베타테스터 후기
콘텐츠 유형: 데모 영상(30초), 얼리버드 혜택 공지, 인플루언서 티저
핵심 문구 방향: "이게 가능해졌어요" / "얼리버드 혜택은 이것입니다"

[D-3 ~ D-1] 긴급성 + 혜택
목표: 마감 임박 심리 자극, 미결정자 전환
전략: 카운트다운 + 혜택 마감 강조 + 반론 처리
콘텐츠 유형: "D-3 남았습니다" 포맷, FAQ 영상, 후기 집중 공개
핵심 문구 방향: "얼리버드 X자리 남았습니다" / "이 혜택은 출시일에 사라져요"

[D-Day] 론칭 + CTA
목표: 첫날 매출 최대화 + 바이럴 유도
전략: 전 채널 동시 오픈 + 구매자 반응 실시간 공유
콘텐츠 유형: 론칭 선언 영상, 구매자 첫 후기 빠르게 공유, 라이브
핵심 문구 방향: "드디어 오픈했습니다" / "첫 {N}명이 이미 시작했어요"
```

---

### Part 2 — 리타겟팅 광고 카피 (페이지 방문자 대상)

> **리타겟팅 원칙:** 처음 보는 사람이 아님. 관심은 있었지만 행동하지 않은 사람.
> 첫 노출 광고와 완전히 다른 언어를 써야 한다.

**단계별 리타겟팅 세그먼트:**

```
[세그먼트 A] 페이지 방문 → 이탈 (인지 단계)
타이밍: 방문 후 1~3일
카피 방향: 다시 상기 + 사회적 증거 강화
예시 헤드라인: "아직 고민 중이세요? 이분들은 이미 시작했습니다"
CTA: 후기 보러 가기

[세그먼트 B] 상세페이지 50% 이상 스크롤 → 이탈 (관심 단계)
타이밍: 방문 후 1~2일
카피 방향: 가장 강한 반론 처리 + 혜택 재강조
예시 헤드라인: "한 가지만 여쭤볼게요 — 무엇이 망설여지세요?"
CTA: 질문하러 가기 / FAQ 보기

[세그먼트 C] 장바구니 담기 → 미결제 (고려 단계)
타이밍: 이탈 후 30분 ~ 24시간
카피 방향: 긴급성 + 마찰 제거 (환불 보장 강조)
예시 헤드라인: "장바구니에 두고 오셨네요 — 혹시 이게 걱정되셨나요?"
CTA: 30일 환불 보장으로 지금 시작

[세그먼트 D] 구매 완료 → 업셀 (전환 후)
타이밍: 구매 후 7~14일
카피 방향: 사용 경험 확인 + 다음 단계 제안
예시 헤드라인: "잘 쓰고 계신가요? 이 기능도 써보셨어요?"
CTA: 다음 단계 보기
```

### 채널별 적용 가이드

```
인스타그램: D-30~D-21 → 카드뉴스/릴스 | D-10~D-Day → 스토리 카운트다운
틱톡:       D-20~D-11 → 비하인드 원테이크 | D-Day → 라이브 + 댓글 반응 영상
유튜브:     D-30 → 롱폼 Pain Point 영상 | D-Day → 론칭 쇼케이스
이메일:     → @email sequence와 연동
커뮤니티:   D-20~D-10 → 관련 커뮤니티 자연스러운 참여 (광고 금지)
```
