# 모션 디렉터 스킬 (Motion Director Skill)

> **담당:** 프로덕트부
> **Tier:** 1 (Gemini Flash) → 복잡한 연출 시 Tier 2
> **Skill ID:** `motion`

## Purpose

텍스트 대본을 입력받아 영상의 컷 전환 시점, 자막 싱크 데이터, BGM 큐 포인트를 자동 생성한다.
Remotion/Kling 렌더링 파이프라인의 입력 데이터를 표준화하는 역할.

## Trigger

- 새로운 쇼츠/영상 대본 완성 시
- `@motion direct` 명령 호출 시
- n8n 유튜브 자동화 파이프라인에서 자동 호출

## Input

```json
{
  "script": "영상에 나올 대본 전문 (읽을 텍스트 전체)",
  "duration_target_sec": 60,  // 목표 영상 길이 (초 단위). 이 시간에 맞게 컷을 분배함

  "style": "shorts | long_form | explainer | ad",
  // shorts: 유튜브 쇼츠/틱톡 스타일 (빠른 편집)
  // long_form: 일반 영상 (여유 있는 편집)
  // explainer: 설명형 영상 (정보 전달 중심)
  // ad: 광고 형식 (Hook→본문→CTA 구조 강제)

  "voice_id": "ElevenLabs에서 설정한 목소리 ID (어떤 목소리로 읽을지)",
  "bgm_mood": "energetic | calm | dramatic | neutral",
  // 배경음악 분위기 선택
  // energetic: 신나는, calm: 차분한, dramatic: 긴장감, neutral: 무난한

  "language": "ko | en | ja",  // 대본 언어 (자막 글자수 제한에 영향)
  "template_id": "디자이너가 미리 만들어둔 템플릿 ID (선택사항)"
}
```

## Output

```json
{
  "timeline": [
    {
      "cut_id": 1,             // 컷 번호 (1번부터 순서대로)
      "start_sec": 0.0,        // 이 컷이 시작되는 시간 (초)
      "end_sec": 3.2,          // 이 컷이 끝나는 시간 (초)
      "text": "이 컷에 표시될 자막 텍스트",

      "visual_directive": "zoom_in | pan_left | static | transition_fade",
      // 화면 연출 지시
      // zoom_in: 화면이 확대되며 들어옴 (강조 효과)
      // pan_left: 화면이 왼쪽으로 이동
      // static: 움직임 없이 고정
      // transition_fade: 페이드 전환

      "emphasis": "none | highlight | bold_text",
      // 자막 강조 방식
      // highlight: 형광펜 효과, bold_text: 두껍게 표시

      "audio_cue": "bgm_start | sfx_whoosh | none"
      // 이 컷에서 재생할 소리
      // bgm_start: 배경음악 시작, sfx_whoosh: 전환 효과음
    }
  ],

  "metadata": {
    "total_cuts": 12,              // 전체 컷 수
    "estimated_duration_sec": 58,  // 예상 완성 영상 길이 (초)
    "word_count": 145,             // 대본 단어 수
    "avg_cut_duration_sec": 4.8    // 평균 컷 길이 (초)
  },

  "subtitle_srt": "SRT 형식의 자막 파일 내용 (영상 편집 프로그램에 바로 삽입 가능)",

  "remotion_props": {
    // Remotion(코드로 영상 만드는 도구)에 넘겨줄 설정값
    "fps": 30,              // 초당 프레임 수 (30이 일반 표준)
    "width": 1080,          // 영상 가로 픽셀 (쇼츠는 세로형이라 1080)
    "height": 1920,         // 영상 세로 픽셀 (쇼츠 9:16 비율)
    "durationInFrames": 1740 // 총 프레임 수 = fps × 초 (30 × 58 = 1740)
  }
}
```

## Process

```
1. 대본 문장 분리 (NLP 기반 문장 경계 탐지)
   // 마침표, 쉼표 등을 기준으로 자연스러운 끊음 지점 자동 탐지

2. 음성 길이 추정 — 언어별 평균 말하기 속도 기준
   └─ ko: 250자/분  // 한국어는 이 속도로 읽는다고 가정
   └─ en: 150단어/분
   └─ ja: 300자/분

3. 목표 시간 내 맞추기 위한 컷 분배 알고리즘
   // duration_target_sec에 딱 맞도록 컷 길이를 자동 조절

4. 스타일별 연출 규칙 적용
   └─ shorts: 평균 컷 3초, 빠른 전환 (시청자 이탈 방지)
   └─ long_form: 평균 컷 7초, 안정적 전환
   └─ ad: Hook(1.5초) + Body + CTA(3초) 구조 (광고 공식 적용)

5. 자막 SRT 생성 (타임코드 자동 계산)
   // 00:00:02,800 --> 00:00:06,500 형식으로 자동 생성

6. Remotion props 계산 (fps × duration)
7. 템플릿 ID 존재 시 해당 템플릿 규격에 맞춰 조정
```

## Example

**Input:**
```json
{
  "script": "월급은 오르지 않는데 물가는 계속 오르고 있습니다. 하지만 정부에서 주는 혜택을 모르는 분들이 너무 많습니다. 오늘 소개할 3가지 숨은 보조금을 확인해보세요.",
  "duration_target_sec": 30,  // 30초짜리 쇼츠로 만들 것
  "style": "shorts",
  "bgm_mood": "energetic",    // 신나는 배경음악으로 집중도 높임
  "language": "ko"
```

## Constraints

- 최소 컷 길이: 1.5초 (가독성 보장 — 더 짧으면 자막을 읽을 수 없음) `[PROD]`
- 최대 컷 길이: 15초 (shorts 기준 10초 — 너무 길면 시청자가 이탈함) `[PROD]`
- SRT 타임코드 정밀도: 밀리초 단위 `[ENG]`
- 자막 한 줄 최대 글자 수: ko 20자, en 40자, ja 25자 `[PROD]`
  // 화면 크기 내에서 읽기 편한 최대치

## Skill Chaining

- **upstream:** Viral Script Skill → motion으로 자동 체이닝
- **downstream:** Remotion 렌더링 → ElevenLabs 음성 합성

## Hard Gates

| ID | 조건 | FAIL 기준 |
|----|------|-----------|
| G1 | 타임라인 | 컷, 자막, BGM/효과음 타이밍이 없음 |
| G2 | 플랫폼 적합성 | Shorts/Reels/TikTok 길이와 화면비를 무시함 |
| G3 | 제작 가능성 | 렌더링 가능한 asset/action 단위로 나뉘지 않음 |

## Soft Findings
